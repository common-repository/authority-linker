The plugin (when activated) adds 2 - 4 links for each post or page automatically and randomly. 

== Description ==
This plugin let's you automatically generate links to authoritative or any sites you want. In plugin settings you generate a list of keywords and URLs for these keywords. When plugin is activated for a certain post or page up to 4 links will be generated automatically with keywords as anchors. This way you don't have to embed links manually and you naturally link to other websites.

Keywords and links can be ex- and imported so that you can use it on several sites

== Installation ==
1. Upload \"authority-linker folder\" to the \"/wp-content/plugins/\" directory.
2. Activate the plugin through the \"Plugins\" menu in WordPress.
